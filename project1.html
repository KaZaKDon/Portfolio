Для веб-сайта предлогающего туристический отдых в горах использовались следующие технологии:

Фронтенд

  • JavaScript - использовался для интерактивности приложения при адаптиве. Замена меню на так называемый гамбургер. При нажатии на который появляется полное меню.
  • HTML/CSS/ - основные технологии веб-разработки для создания структуры, стилей веб-страниц.
  • Flex-box - технология при адаптивной верстке.

Процесс разработки

Разработка и развертывание сайта предлогающего туристический отдых в горах включили в себя несколько ключевых этапов:

  • Проектирование. На этом этапе мы создали макет сайта, включавший - выбор цветовой палитры, шрифтов, изображений и других элементов дизайна.
  • Разработка. Написали код сайта, используя выбранные технологии (HTML, CSS, JavaScript ). Удостоверились, что код хорошо структурирован, легок для чтения и поддержки.
  • Развертывание. После того как сайт был завершен. Был перемещен в репозиторий на Github. От куда был собран, развернут и опубликован
